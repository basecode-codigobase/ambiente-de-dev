# ambiente-de-dev
Os dois metodos para organizar o ambiente de desenvolvimento tanto no Windows, quanto no Linux. Explicado passo-a-passo.

## Checklist final do Linux

Marque o que já deu certo:

- [ ] Git funcionando
- [ ] Chave SSH criada
- [ ] Chave SSH adicionada no GitHub
- [ ] Teste com GitHub funcionando
- [ ] Node.js funcionando
- [ ] npm funcionando

---

## Checklist final do Windows

Marque o que já deu certo:

- [ ] WSL 2 instalado
- [ ] Ubuntu 24.04 instalado
- [ ] Git funcionando
- [ ] Chave SSH criada
- [ ] Chave SSH adicionada no GitHub
- [ ] Teste com GitHub funcionando
- [ ] Node.js funcionando
- [ ] npm funcionando
- [ ] VS Code instalado no Windows
- [ ] VS Code abrindo com `code .` no WSL
